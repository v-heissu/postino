# Postino

API pagination aggregator - aggrega risultati da API paginate in un unico JSON.

## Uso

1. Inserisci l'endpoint URL
2. Inserisci il Bearer Token
3. Modifica il body JSON con i parametri desiderati (limit, offset, fields, etc.)
4. Clicca "Fetch All Pages"
5. Copia o scarica il risultato aggregato

## Deploy su Railway

[![Deploy on Railway](https://railway.app/button.svg)](https://railway.app/template)

1. Collega il repository a Railway
2. Railway rileverà automaticamente Node.js e farà il deploy

## Sviluppo locale

```bash
npm install
npm start
```

L'app sarà disponibile su http://localhost:3000
